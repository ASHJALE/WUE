# WUE (Wood U Estimate)

**AI-Assisted Furniture Material Estimation and Quotation System**

WUE (Wood U Estimate) is a web-based capstone project designed to help furniture manufacturers, business owners, and customers estimate furniture materials and generate cost quotations. The system uses artificial intelligence to recognize furniture from uploaded images and automatically estimates the required materials, production cost, and quotation.

---

## Features

- 🪑 Furniture image recognition
- 📦 Material estimation
- 📝 Bill of Materials (BOM)
- 📊 Inventory management
- 💰 Cost estimation
- 📄 PDF quotation generation
